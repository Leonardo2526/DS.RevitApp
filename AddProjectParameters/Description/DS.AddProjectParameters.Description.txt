Описание:
1. Данная программа выполняет пакетную загрузку параметров из файла общих параметров во все файлы *.rvt в указанной директории;
2. После загрузки все файлы *.rvt сохраняются;
3. Все вновь создаваемые параметры применяются к типу категорий "Модели" и являются параметрами "Типа". 
4. Все результаты выполненных действий записываются в папку "Log" в корне директории, указанной для файлов *.rvt.


Инструкция по пользованию:
1. Создать ФОП *.txt, на основе которого будет выполняться загрузка новых параметров проекта;
2. Создать *.txt файл с путями к проектам *.rvt;
3. Указать *.txt файл с путями к проектам *.rvt;
4. Указать *.txt файл ФОП;
5. Выбрать группу и параметры, которые требуется загрузить;
6. Нажать "Apply" или "Apply all".
