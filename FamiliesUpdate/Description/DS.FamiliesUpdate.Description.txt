Описание:
1. Данная программа выполняет пакетную загрузку семейств во все файлы в указанной директории:
2. После загрузки все файлы *.rvt сохраняются;
3. Если в файле *.rvt уже присутствует загружаемое семейство, то при наличии отличающихся параметров и их значений оно обновляется.
	В противном случае оно не загружается. 
4. Все результаты выполненных действий записываются в папку "Log" в корне директории, указанной для файлов *.rvt.


Инструкция по пользованию:
1. Указать исходную директорию, в которой содержатся папки с *.rvt файлами;
2. Указать исходную директорию, в которой содержатся семейства с *.rfa файлами;
3. При необходимости применить фильтры.
4. Перед загрузкой семейств рекомендуется воспользоваться плагином по пакетному 
	обновлению версий файлов семейств "ADNPlugin-FileUpgrader".
