Описание:
1. Данная программа выполняет пакетную загрузку новых параметров в файла общих параметров;
2. Все результаты выполненных действий записываются в папку "Log" в корне директории, указанной для файлов *.rvt.


Инструкция по пользованию:
1. Создать ФОП *.txt, на основе которого будет выполняться загрузка новых параметров;
2. Создать *.txt файл с именами новых параметров;
3. Указать ФОП *.txt
4. Указать *.txt файл с именами новых параметров;
5. Выбрать группу или создать новую, в которую требуется загрузить новые параметры;
6. Нажать "OK".
